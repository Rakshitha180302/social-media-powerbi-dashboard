Social Media Analytics Dashboard (Power BI)

A clean and interactive Power BI dashboard analyzing cross-platform social media performance across engagement, audience, influencers, campaigns, competitors, hashtags, and summary overview.


Pages Included

1. Engagement – Daily trends, platform breakdown, ER%
2. Audience – Age/gender matrix, quality analysis
3. Influencer – Top influencer scatter, ER drivers
4. Competitor – KPI benchmarks, engagement gaps, competitor comparisons
5. Hashtag – Usage vs engagement, event markers, influencer toggle
6. Overview – KPIs, summary tables, paginated narrative


Built With

- Power BI (.pbix)
- DAX Measures
- Interactive slicers, toggles
- Smart labels, conditional formatting
- Custom annotations and tooltips
